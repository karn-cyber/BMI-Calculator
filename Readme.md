# BMI Calculator
# A modern, responsive Body Mass Index (BMI) calculator built with HTML, JavaScript, and styled with Tailwind CSS. This application allows users to calculate their BMI by entering their height and weight, and provides clear feedback with BMI categories.
# 
# Features
# Responsive Design: Works seamlessly on both desktop and mobile devices.
# Input Validation: Ensures valid height and weight inputs with clear error messages.
# BMI Categories: Displays the calculated BMI along with its category (Underweight, Normal Weight, Overweight, Obese) with color-coded feedback.
# Modern UI: Clean and professional interface using Tailwind CSS for styling.
# No Form Submission: Uses button click event to avoid form-related issues in certain environments.
# Technologies Used
# HTML5: For the structure of the web page.
# JavaScript: For BMI calculations and dynamic content updates.
# Tailwind CSS: For responsive and modern styling via CDN.
# Setup
# Clone or Download the Project:
# Clone this repository or download the index.html file.
# No Additional Dependencies:
# The project uses Tailwind CSS via a CDN, so no local installations are required.
# Open the Application:
# Open index.html in a web browser to run the BMI Calculator.
# Usage
# Enter Height:
# Input your height in centimeters (cm) in the provided field.
# Enter Weight:
# Input your weight in kilograms (kg) in the provided field.
# Calculate BMI:
# Click the "Calculate BMI" button to see your BMI and category.
# View Results:
# The result will display your BMI value and its corresponding category (Underweight, Normal Weight, Overweight, or Obese) with color-coded feedback.
# If invalid inputs are provided, an error message will be shown.
# BMI Categories
# Underweight: Less than 18.5 (Yellow)
# Normal Weight: 18.5 - 24.9 (Green)
# Overweight: 25 - 29.9 (Orange)
# Obese: 30 or greater (Red)
# File Structure
# index.html: Contains the HTML structure, JavaScript logic, and Tailwind CSS integration for the BMI Calculator.
